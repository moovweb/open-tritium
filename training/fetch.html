Fetch

fetch() is a function that allows you to grab something (for example a node or some text), and pass it into a variable.

Fetching

To fetch some data, just scope into the parent node:

$("./div") {

}

Then set a variable using the dollar sign syntax:

$("./div") {
  $var = 
}

Finally, use the fetch() function to grab whichever item you require within the <div>:

$("./div") {
  $var = fetch("./a")
}

To grab some text from a node, just use text() instead of the path to the node.

Example

<div>
  <a href="http://google.com">Google</a>
</div>

Take the above HTML as our example.

The following Tritium:

$("div") {
  $var1 = fetch("./a")
}

will set the variable $var1 to "<a href="http://google.com">Google</a>".

Whereas the following Tritium:

$("div") {
  $var1 = fetch("./a/text()")
}

will set the variable $var1 to just "Google".
